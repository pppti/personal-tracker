包材管理系统 - 公司内网部署说明

一、为什么今天打不开

昨天网页能打开，是因为本机运行着 python app.py。
今天打不开，是因为电脑重启或窗口关闭后，后台服务没有继续运行。
数据库文件没有丢，位置是：

  data\skincare.db

二、公司电脑怎么用

推荐做法：把整个 webapp 文件夹复制到公司电脑，例如：

  D:\vs-claude\webapp

公司电脑需要安装 Python 3.12 或更高版本。
然后双击：

  start.bat

启动后，本机访问：

  http://127.0.0.1:5000

三、公司内网多台电脑怎么访问

1. 选一台公司电脑作为主机，运行 start.bat。
2. 在这台主机上查看 IP 地址，例如 192.168.1.100。
3. 其他同一个公司内网的电脑访问：

  http://192.168.1.100:5000

注意：127.0.0.1 只代表“当前这台电脑自己”，不能给其他电脑访问。
其他电脑必须用主机的内网 IP 地址。

四、开机自启

在公司主机上，右键 PowerShell，选择“以管理员身份运行”，执行：

  cd /d D:\vs-claude\webapp
  .\install_service.ps1

安装后，电脑开机或用户登录时会自动启动。

五、防火墙

如果主机自己能打开 http://127.0.0.1:5000，
但其他电脑打不开 http://主机IP:5000，
通常是 Windows 防火墙拦截了 5000 端口。

解决方法：

  Windows 安全中心 -> 防火墙和网络保护 -> 高级设置
  入站规则 -> 新建规则 -> 端口 -> TCP -> 5000 -> 允许连接

六、数据备份

所有业务数据都在：

  data\skincare.db

备份这个文件即可。
迁移到公司电脑时，如果要保留当前数据，也要把 data 文件夹一起复制过去。

七、重要提醒

如果家用电脑和公司电脑不在同一个网络，公司电脑不能直接访问家用电脑上的 127.0.0.1。
要在公司使用，最稳妥的方法是把 webapp 部署到公司内网的一台电脑上。
